# Руководство по обновлению

## Проблема

Если вы переустановили программу, но новые функции не появились, это означает, что старая версия все еще используется.

## Решение 1: Автоматическое обновление (рекомендуется)

```bash
# Перейдите в директорию проекта
cd UbuntuAutoUpdater

# Сделайте скрипт обновления исполняемым
chmod +x update.sh

# Запустите обновление
sudo ./update.sh
```

Скрипт автоматически:
- Удалит старую версию
- Установит новую версию v2.0.0
- Проверит корректность установки

## Решение 2: Ручное обновление

### Шаг 1: Удалите старую версию

```bash
sudo rm /usr/local/bin/aupdate
```

### Шаг 2: Установите новую версию

```bash
cd UbuntuAutoUpdater
chmod +x install.sh
sudo ./install.sh
```

### Шаг 3: Проверьте установку

```bash
sudo aupdate
```

Вы должны увидеть меню с 36 опциями и версию v2.0.0.

## Решение 3: Полная переустановка

Если проблема сохраняется:

```bash
# 1. Удалите старую версию
sudo rm /usr/local/bin/aupdate

# 2. Удалите старую директорию (если есть)
cd ~
rm -rf UbuntuAutoUpdater

# 3. Клонируйте заново
git clone https://github.com/Fixcat/UbuntuAutoUpdater.git
cd UbuntuAutoUpdater

# 4. Установите
chmod +x install.sh
sudo ./install.sh

# 5. Запустите
sudo aupdate
```

## Проверка версии

После обновления проверьте версию:

```bash
sudo aupdate
```

В заголовке должно быть:
```
╔════════════════════════════════════════╗
║   Ubuntu System Updater v2.0.0         ║
╚════════════════════════════════════════╝
```

И меню должно содержать 36 опций (0-36).

## Возможные проблемы

### Проблема: Команда aupdate не найдена

**Решение:**
```bash
# Проверьте, установлена ли программа
ls -la /usr/local/bin/aupdate

# Если файл не существует, установите заново
cd UbuntuAutoUpdater
sudo ./install.sh
```

### Проблема: Старая версия все еще запускается

**Решение:**
```bash
# Проверьте, какой файл используется
which aupdate

# Если показывает другой путь, удалите его
sudo rm $(which aupdate)

# Затем установите заново
cd UbuntuAutoUpdater
sudo ./install.sh
```

### Проблема: Меню показывает только 4 опции

Это означает, что используется старая версия (v1.0.0).

**Решение:**
```bash
# Принудительно удалите старую версию
sudo rm -f /usr/local/bin/aupdate

# Установите новую
cd UbuntuAutoUpdater
sudo ./update.sh
```

## Проверка успешного обновления

После обновления вы должны увидеть:

✅ Версия v2.0.0 в заголовке
✅ 36 опций в меню (0-36)
✅ Разделы: ОСНОВНОЕ, УПРАВЛЕНИЕ ПАКЕТАМИ, ОБСЛУЖИВАНИЕ, ИНФОРМАЦИЯ, и т.д.
✅ Новые функции: установка пакетов, резервное копирование, статистика

## Дополнительная помощь

Если проблема сохраняется:

1. Проверьте содержимое файла:
```bash
head -20 /usr/local/bin/aupdate
```

Первая строка должна содержать:
```bash
#!/bin/bash
```

Вторая строка:
```bash
# Ubuntu System Updater v2.0
```

2. Проверьте размер файла:
```bash
ls -lh /usr/local/bin/aupdate
```

Размер должен быть около 20-25 KB (новая версия намного больше старой).

3. Если размер маленький (около 5 KB), это старая версия - удалите и установите заново.
